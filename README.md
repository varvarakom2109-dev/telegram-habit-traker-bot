# telegram-habit-traker-bot
Telegram-бот для трекинга привычек.

## Возможности
- Добавление привычек.
- Напоминания
- Отметка выполнено / не выполнено
- Серии
- История и статистика
- Редактирование привычек

## Установка
'''bash
git clone https://github.com/varvarakom2109-dev/telegram-habit-traker-bot.git
cd habit-tracker-bot
pip install -r requiments.txt

## Настройка
Создайте config.py на основе config.example.py и встатьте токен бота.

## Запуск
python main.py
